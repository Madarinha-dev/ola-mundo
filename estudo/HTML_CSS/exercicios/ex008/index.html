Criando uma Tabela (Table)

Usando:
<table> essa tag ela sevre para criar uma tabela o problema
é que ela é um pouco complexa ate, vai está sua estrutura. 

        <table>
            <caption>Subtitulo</caption>
            <thead> <tr><th>Defini os intems</th></tr></thead>
            <tfoot><tr><th>Defini os intems</th></tr></tfoot>
            <tbody>
                <tr><td>10</td></tr>
                <tr><td>15</td></tr>
            </tbody>
        </table>

Essa é a estrutura pra fazer uma tabela 

     <table> = Serve para identificar o inicio da tabela 

     <Caption> = serve como o tema da tabela 

     <thead e o tfoot> = cabeçalho e roda pe da tabela 

     <tr> = Ele cria uma linha da tabela 

     <th> = Ele defini os intems da tabela 

     <tbody> = Aqui vocês ja sb é o corpo da Tabela onde fica as informações / dados 

     <td> = Defini a informação / dados da tabela
